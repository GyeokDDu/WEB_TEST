HTML

HTML (Hypertext Markup Language)이 표준입니다. 웹 페이지 및 웹 만들기를위한 마크업 언어입니다. 웹 브라우저는 웹 서버에서 HTML 문서를 받거나 로컬 저장소에서 멀티미디어 웹 페이지로 렌더링합니다. HTML은 웹 페이지의 구조를 의미론적으로 설명하여 원래 문서 모양에 대한 신호를 포함합니다.

HTML 요소는 HTML 페이지의 구성 요소입니다. HTML 구조를 사용하면 이미지 및 대화 형 양식과 같은 기타 개체가 렌더링 된 페이지에 삽입 될 수 있습니다. 그것은 창조하는 수단을 제공합니다. 제목, 단락, 목록, 링크, 인용문 및 기타 항목과 같은 텍스트에 대한 구조적 의미를 표시하여 구조화 된 문서를 작성하는 수단을 제공합니다. HTML 요소는 태그로 표시됩니다.
